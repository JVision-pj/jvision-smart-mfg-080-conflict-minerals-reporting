# Conflict Minerals Reporting智慧站

管理供應鏈衝突礦產（3TG）來源申報，符合客戶與國際負責任礦產倡議要求。

- 原始系統：Conflict Minerals Reporting（衝突礦產申報管理）
- 產業分類：智慧製造｜品質管理
- 使用部門：採購部、法規事務部、品保部
- 工作流程：問卷發送→供應商填報→數據驗證→冶煉廠比對→報告彙總提交
- AI 應用：問卷回覆完整度自動檢核、風險冶煉廠標記
- 主要 KPI：供應商回覆率、合規冶煉廠比例、申報準時率
- 代表產品：Assent Compliance、RCS Global、以RMI標準範本客製化開發為主
